# Giga Code VSCode Skills

Коллекция скиллов для плагина Giga Code VSCode, основанных на методологии SPDD (Structured Prompt-Driven Development).

## Доступные скиллы

### spdd-reasons-canvas
Генерация структурированного REASONS Canvas перед кодированием.

**Когда использовать:**
- Перед началом реализации новой функции
- При получении крупного требования
- Для согласования подхода

### spdd-code-generation
Генерация кода по REASONS Canvas.

**Когда использовать:**
- При реализации по готовому Canvas
- Для соблюдения Operations, Norms и Safeguards

### spdd-api-testing
Генерация cURL-based API тестов.

**Когда использовать:**
- После создания API эндпоинтов
- Для тестирования граничных случаев
- Перед релизом

### spdd-domain-analysis
Доменный анализ и извлечение ключевых концепций.

**Когда использовать:**
- Перед созданием Canvas
- Для понимания существующей архитектуры
- При выявлении рисков и зависимостей

### spdd-story-breakdown
Разбиение требований на пользовательские истории по принципу INVEST.

**Когда использовать:**
- Когда есть большое требование
- Для планирования спринта
- При декомпозиции эпиков

## Интеграция с VSCode

Скиллы доступны через команды плагина Giga Code:
- `Giga: Generate REASONS Canvas`
- `Giga: Generate Code from Canvas`
- `Giga: Generate API Tests`
- `Giga: Analyze Domain`
- `Giga: Breakdown Story`

## Принципы работы

Скиллы преобразуют SPDD-методологию в практические, готовые к использованию инструменты внутри редактора кода. Каждый скилл:

- Сохраняет полную совместимость с оригинальными SPDD скиллами
- Адаптирован для работы в контексте VSCode
- Поддерживает горячие клавиши и интеграцию с UI
- Использует те же шаблоны и принципы, что и в `.agents/skills`
